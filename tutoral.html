Langkah 1: Mengenal HTML

HTML (HyperText Markup Language) adalah bahasa markah yang digunakan untuk membuat struktur dan konten sebuah halaman web.

Berikut langkah-langkah sederhana menggunakan HTML:

  1. Buka teks editor favorit Anda.
  2. Tulis tag pembuka HTML: <html>.
  3. Tambahkan tag pembuka dan penutup <head> dan </head>.
  4. Tambahkan tag pembuka dan penutup <body> dan </body>.
  5. Isi body dengan elemen-elemen HTML seperti <h1>, <p>, dll.
  6. Simpan file dengan .html.
  7. Buka file dengan browser untuk melihat hasilnya.

Langkah 2: Mengenal CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tata letak dan tampilan dari elemen-elemen dalam halaman web.

Berikut langkah-langkah menggunakan CSS:

  1. Tambahkan elemen <link> untuk merujuk ke file CSS.
  2. Tulis aturan CSS untuk mengatur tampilan elemen-elemen.
  3. Simpan file CSS dengan ekstensi .css.
  4. Tambahkan kelas CSS ke elemen HTML.

Langkah 3: Membuat Halaman Web Sederhana

Gabungkan HTML dan CSS untuk membuat halaman web sederhana dengan tata letak dan desain yang menarik.

Anda dapat menambahkan elemen-elemen seperti <header>, <footer>, <nav>, dan lain-lain.
